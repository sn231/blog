第一，**望月为什么不继续追王林？**

不是因为它追不上就放弃，也不是因为它突然心善，而是到了**罗天南北域交界处**后，望月本能地停住了。

原文逻辑大概是：

王林肉身被古神一指毁掉后，元神化作太古雷龙，卷着储物袋、仙卫和半截望月骨逃。因为只剩元神，他速度反而比有肉身时快很多，再加上星罗盘，勉强和望月保持距离。

但这只是“没被立刻追上”，不是彻底逃脱。真正让他活下来的关键是：

**望月追到罗天南北域交界处时，停了，不敢跨过去。**

后面还写到，望月眼中有犹豫，像是记忆里有人警告过它：**不要踏入前方**。所以这不是王林打退了它，而是那片区域对望月有某种古老禁忌、威慑或规则限制。

王林自己当时也不明白，只是猜：

> 这里是不是有什么让望月害怕的存在？

所以最准确的说法是：**望月不是不想追，而是不敢越过罗天南北域交界。具体是谁警告过它，或者那里到底有什么，作者在当时没有马上讲透，只留下了一个伏笔式解释。**

第二，**古神修为体系和人类修士怎么对应？**

古神和人类修士不能完全一一对应。人类修士主要修**元力、意境、规则、本源**；古神主要修**肉身、古神之星、古神之力**。所以只能粗略对照。

人类修士大线是：

**第一步：**
凝气 → 筑基 → 结丹 → 元婴 → 化神 → 婴变 → 问鼎

**第二步：**
阴虚 → 阳实 → 窥涅 → 净涅 → 碎涅 → 天人五衰

**第三步：**
空涅 → 空灵 → 空玄 → 空劫，再往上就是更高层次的大能体系。

古神则是：

**一星到九星古神。**
星越多，肉身越强，能动用的古神神通越多。王族古神还要比普通古神强一大截。

粗略对应可以这么看：

| 古神星级  | 大概对应人类修士                         |
| ----- | -------------------------------- |
| 一星、二星 | 没有稳定对应，属于较弱古神层次                  |
| 三星古神  | 大约能威胁化神、婴变层次；王林三星本尊加分身，可战最弱婴变    |
| 四星古神  | 关键门槛，原文明确说三星升四星后，大致相当于问鼎修士       |
| 五星古神  | 超越问鼎一个大层次，约等于第二步初中段，至少不是普通问鼎能碰的  |
| 六星古神  | 第二步高层级，接近或压过很多碎涅修士，但仍未真正等同第三步    |
| 七星古神  | 完整七星肉身，已经可以和第三步空涅初期一战            |
| 八星古神  | 极其恐怖，尤其王族八星，普通第三步也要忌惮            |
| 九星古神  | 古神顶峰层次，不能简单用普通修士境界衡量，基本属于星空级灾难存在 |

但这里要注意一个坑：

**三星古神不等于婴变，四星古神不等于普通问鼎，五星古神也不等于固定的窥涅。**

因为古神强在肉身和爆发，人类强在法术、规则、法宝、意境。真打起来还要看血统、神通、法宝、状态、是否王族。

比如王林的本尊是古神肉身，但分身又有人类修士体系。他合体后战力会跳级，所以不能拿王林的战绩直接当“古神星级=人类境界”的标准。

最稳的记法是：

**四星古神≈问鼎门槛。
五星古神＞问鼎，进入第二步层次。
七星古神≈可战第三步空涅初期。
八星、九星古神就已经不是普通修士体系能舒服对照的怪物了。**
